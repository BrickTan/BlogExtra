Clase 4: Agregar y eliminar elementos del DOM

En esta clase aprendimos a agregar y eliminar elementos dinámicamente con JS. A nuestro HTML y css de la clase le agregamos un input para escribir la tarea y un botón ara agregarla a la lista; además de que si seleccionamos una tarea se borra. Primero llamamos el elemento de form del HTMl y le asignamos un evento en nuestro script de Javascript; y con una nueva función llamada "Handle Submit" en donde se agrega la tarea agregamos un event.preventDefault el cual evita que la página se actualice y se pierda la información.
Vimos una nueva función que se llama querySelector que busca con un selector parecido a css un elemento ya sea por id, class o el nombre su tag.
Hicimos una nueva función para agregar una tarea con un parámetro, en donde creamos elementos con la función document.createElement. Por último creamos una nueva función llamada deleteTask en donde se le agrega un event.target para identificar el elemento seleccionado y después le añadimos una función llamada .remove() que lo elimina.

Algo que me causo mucho conflicto en esta tarea fue que se crearon muchas funciones y cada una hacía una cosa diferente pero también dependían una d ela otra para poder funcionar. También en un inicio no entendia como funcionaba la función del querySelector pero quedó un poco más claro al hacer la tarea, ya que se refiere al elemento que se quiere seleccionar y también se puede utilizar el querySelectoAll que son todos los elementos que tengas ese tag, id o clase.
