Clase 3: To do 01

Esta clase hicimos una lista de cosas por hacer, con opción a un dark mode en su diseño y que cuando seleccionara una tarea se tachara. Primero se desarrolló un HTML y su css, en el cual se puso el diseño normal y un diseño extra con una referencia de clase para cuando fuera dark mode. Repasamos el llamado de referencias de HTML a JS y la asignación de eventos para que ocurriera cierta acción. Algo nuevo que vimos fue que en el parámetro de una función se puede poner un evento para utilizar otras propiedades como target, para aplicar cierta acción que se requiera al elemento que se está seleccionando y la función toggleque funciona como switch.

Recuerdo que en esta clase algo que no entendí fue como funcionaba la función toggle, ya que no sabía como es que el sistema sabía que había cambiado, pensaba que se tenía que definir que ahora estaba de otra forma para que la anterior estuviese disponible, pero entedí que funcionaba como switch de prendido y apagado automáticamente. Fue también muy interesante utilizar la función classList que te permite agregar o quitar una clase a un elemento, de esta forma se implemento el dark mode, y el check.
