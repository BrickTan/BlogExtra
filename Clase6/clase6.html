Clase 6: To Do Render

Vimos un nuevo método para hacer toda la funcionalidad de la clase pasada, ya que no era la mejor forma ya que no se querían modificar las cosas dentro del HTML como lo estábamos haciendo hasta ahora. En Javascript se pueden meter funciones dentro de otras funciones, esto con la finalidad de poder usar variables y no volver a declararlas. Y así creamos una nueva función llamada Render(). En la función de addTask que habíamos utilizado las clases pasadas borramos todo y agregamos solamente una función nueva que se llama .push que agrega el elemento que se le indica.
En la función Render agregamos a un elemento de la lista de tareas una función que se llama innerHTML que se refiere al HTML que tiene este elemento, de esta forma se pueden construir templates de HTML y por cada elemento de la lista se le agregó este Template. También vimos una nueva sintaxis para agregar variables dentor de los templates, la cual es $(variable) Por último vimos Data attributes que se utilizan para la comunicación del JS y el HTML para asignar valores y después poderlos obtener, como por ejemplo, un index para poder utilizarlo después como en la función de Delete task para saber que elemento eliminar. Con Dataset llamamos a todos los data attributes del HTML. Y en lugar de remove utilizamod Splice que acepta parámetros, entonces ponemos el dataset agregado en una nueva variable y un 1, que significa eliminar ese elemento del index.
